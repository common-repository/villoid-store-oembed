Share Products from Villoid Stores to your Wordpress blogs.

== Description ==

It enables sharing via oEmbed for your villoid shop under store_name.villoid.com

== Installation ==

Installing "Villoid Store oEmbed" can be done either by searching for "Villoid Store oEmbed" via the "Plugins > Add New" screen in your WordPress dashboard, or by using the following steps:

1. Download the plugin via WordPress.org
1. Upload the ZIP file through the 'Plugins > Add New > Upload' screen in your WordPress dashboard
1. Activate the plugin through the 'Plugins' menu in WordPress
